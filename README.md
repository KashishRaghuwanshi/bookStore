**Online Book Store** is a full-stack web application designed to provide users with a seamless and interactive platform to browse, search, and purchase books online. The project is built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js) with **Tailwind CSS** for responsive and modern UI design.

* **MongoDB**: A NoSQL database used to store and manage book details, user information, and orders efficiently.

* **React.js**: A JavaScript library for building fast and dynamic user interfaces.
* **Tailwind CSS**: A utility-first CSS framework used for creating sleek and responsive UI components.
* **Node.js**: A JavaScript runtime environment used to run the backend server and handle requests.
